Идеи расположены в порядке убывания приоритетности.
 
1. Приложение для квестов
Черновой дизайн: https://github.com/maxbach/MainVkProject/blob/master/20161012_103128.jpg
- Приложение включает в себя множество интересных квестов. Есть вариант выполнить это, как биржа квестов, где есть продавцы и покупатели. 
- Квесты представляют собой задания на определенную тему, которые заставляют участника путешествовать и искать ответы на вопросы. Например, ответы можно узнать, если посетить какой-то памятник или прогуляться в парке
- ЦА: активные люди, которые ищут путешествий в родном городе, которые хотят узнать что-то нового про родной город, которые хотят интересно и недорого провести время, просто гуляя по интересным местам. 16-45 лет
 
2. Умный помощник в планирвоании дел
Черновой дизайн: https://github.com/maxbach/MainVkProject/blob/master/20161012_103321.jpg
- Приложение, которое поможет разложить все дела "по полочкам"
- Пользователь сначала обозначает сферы своей жизни, напрмиер универ, спорт и другое
- Для каждой сферы выделяется доска, куда пользователь будет записывать все свои задания, дела
- Одна из главных идей в том, чтобы пользователь научился разбивать свое задания на множество мелких, для этого будет предусмотрена иерархия дел
- Также удет функция, когда пользователь сможет указывать длительность дел, чтобы было наглядно видно, сколько времени займет то или иное задание
- Конечно, можно задавать дедлайн, напоминания и другое
- Когда пользователь разбил свои дела на мелкие задания, далее он распределяет их по дням. Будет предусмотрена функция составления расписания по дням и пользователь дальше может передвигать задания по дням
- Таким образом, будет готово расписание своего дня, далее все зависит от пользователя
- В планах, внедрение в программу инетерсных идей тайм-менеджмента, которые считаются эффективными.
- ЦА: школьники, студенты или рабочие, у которых много сфер жизни, за которыми трудно успевать. Прилоежние поможет им разгребсти всю кучу дел и разложить по полочкам.

6. Приложение, которое содержит настольные игры 

- Большинство настольных игр не требует игровых полей, особых фигурок и других сложных изделий. В основном, нужны карточки, какое-то игровое поле, несложное
- Идея состоит в том, чтобы игроки соединялись по Bletooth, чтобы программа уже распределяла материал между игроками
- Например, для игры в мафию нужны карточки. Игроки соединяются друг с другом, программа рассылает карточки игрокам, заменяет ведущего. И таких примеров, сотни
- Можно создать базу данных таких игр для компании.
